 # Calculadora e lista de exercicios
 

  Calculadora simples, e lista de exercicios com calculadoras especificas

  ## Funcionalidades 

  - **Operaçoes basicas**: pode realizar somas, subtraçoes, multiplicaçoes e divisao
  - **suporte a numeros decimais**: E possivel realizar operaçoes com ate 2 casas decimais ou mais
  - **Validaçao de entrada**: E garantido a entrada de apenas opçoes validas 
  - **tratamento de divisao por zero**: E possivel validar o erro de divisao por zero


  ---


  ## Como usar

  1. Clone ou baixe o codigo fonte/repositorio da calculadora que deseja utilizar.
  2. Abra o terminal de comando ou o prompt ate a pasta da calculadora
  3. Utilize este comando abaixo para restaurar as dependencias do projeto.
  ```

  dotnet restore

 ```
  4. Em seguida compile a soluçao utilizando o comando 

  ```
	
dotnet build --configuration release

```

5. para executar o projeto compilando em tempo real
```
dotnet run --project (calculadora que deseja utilizar)


```
## Requisitos 
- .NET SDK (.NET 5.0 ou superior para a execuçao do projeto)
